About Site

Features
  • Angular: This site was built using Angular CLI. Each component was created using Angular Material and Angular Themes
  • Express:While this site is being hosted by Firebase, Webpack and Express handed the routing
  • Firebase:Site is hosting, Google Authentication, and Database for the blog are on Firebase
  • Bootrap:For icons, and general grid/font help
Progressive Web App
  • This site currently has a lighthouse score of: 89
  • I am hoping to improve this by using text compression and adding more accessability features.
  • It does include a service worker and app manifest for offline use.
